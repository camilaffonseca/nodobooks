# TextField

### features:

- Styled TextField to use anywhere!

### props

- label: Top name in input.
- name: Unic name. Use by react-hook-form.
- register: heact-hook-form register function.
- placeholder
- error: Error message string.
- message: If passed, replace error with.
- disabled: Boolean to disable input.
- type: HTML input types.

## Basic usage

```
<ThemeProvider>
  <TextField ...props />
</ThemeProvider>
```
