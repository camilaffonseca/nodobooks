# RatingBar

### features:

- Styled RatingBar to use anywhere!
- Auto calculate percentage inside the bar.

### props

- percent: Progress percent. (0 / 100)

## Basic usage

```
<ThemeProvider>
  <RatingBar percent={76} />
  <br />
  <RatingBar percent={21} />
  <br />
  <RatingBar percent={87} />
  <br />
  <RatingBar percent={100} />
  <br />
  <RatingBar percent={0} />
</ThemeProvider>
```
